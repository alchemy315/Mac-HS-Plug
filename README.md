# 炉石传说 一键拔线 (macOS) ⚡️

这是一个专为 macOS 玩家打造的炉石传说（尤其是酒馆战棋模式）辅助工具。它通过静默调用 **Little Snitch** 的命令行接口，实现对游戏网络连接的秒级切断与恢复，帮助你优雅地跳过冗长的战斗动画。

## ✨ 核心特性

- 🔍 **智能监控**：实时分析流量，自动锁定对局服务器 IP（无域名解析连接）。
- ⚡️ **极速响应**：毫秒级断网与重连，确保跳动画成功且不掉出游戏。
- 🖱 **三位一体控制**：
  - **全局快捷键**：自定义按键，随时触发。
  - **悬浮窗按钮**：始终置顶，通过红/黄/绿颜色实时反馈对局状态。
  - **菜单栏图标**：常驻右上角，方便快速启停监控。
- 🔐 **安全加密**：Sudo 密码加密存储于系统 Keychain，不弹出终端窗口。
- 📊 **可视化日志**：详细的彩色日志记录，操作状态一目了然。

## 🛠 运行前提

1. **系统要求**：macOS 12.0 或更高版本。
2. **必要软件**：必须安装并激活 **[Little Snitch](https://www.obdev.at/products/littlesnitch/index.html)**。

## 🚀 安装与配置流程

### 第一步：开启 Little Snitch 命令行访问权限
这是程序能够自动修改规则的前提，请务必执行：
1. 打开 Little Snitch 软件。
2. 点击屏幕左上角菜单栏 `Little Snitch` -> `Settings...`。
3. 选择 `Security`（安全）标签页。
4. **勾选** `Allow access via Terminal`（允许通过终端访问）。

### 第二步：安装本程序
1. 在本仓库的 [Releases](https://github.com/your-username/your-repo/releases) 页面下载最新的 `.dmg` 文件。
2. 双击打开并将“一键拔线”拖入 `Applications`（应用程序）文件夹。

### 第三步：软件初始化
1. 打开软件，进入 **【系统设置】** 页面。
2. **路径设置**：点击浏览或自动检测 Little Snitch 与炉石传说的路径。
3. **权限确认**：在输入框中输入你的 Mac 开机密码，点击 **“保存密码”**（该操作只需执行一次）。
4. 点击 **“确认权限”**，在弹出的系统钥匙串请求中选择 **“始终允许”**。

## 🎮 使用指南

1. **开启监控**：在【拔线】页面打开“开启对局监控”。
2. **进入游戏**：启动炉石传说并进入酒馆战棋。当战斗开始后，你会看到软件底部的指示灯或悬浮窗变绿，提示“已检测到对局 IP”。
3. **执行拔线**：在战斗动画期间，按下自定义的全局快捷键（或点击悬浮窗/菜单栏按钮）。
4. **动画跳过**：游戏会瞬间重连并回到酒馆，战斗动画已跳过。

## ⚠️ 常见问题：提示“应用已损坏”怎么办？

因为本软件属于开源未签名应用，macOS 的 Gatekeeper 安全机制可能会拦截。如果你在运行软件时遇到**“应用已损坏，无法打开”**的提示，请按以下步骤解决：

1. 将 App 从 DMG 中拖拽到你 Mac 的 **“应用程序 (Applications)”** 文件夹中。
2. 打开 Mac 自带的 **“终端 (Terminal)”**。
3. 复制并粘贴以下命令，然后按下回车：
   sudo xattr -r -d com.apple.quarantine /Applications/一键拔线.app
   *(注意：如果你的应用名字不是“一键拔线”，请自行替换路径)*
4. 终端会要求你输入 Mac 的开机密码（输入时不会显示密码字符，输入完直接回车即可）。
5. 再次去“应用程序”里双击打开软件，就可以正常运行了！
